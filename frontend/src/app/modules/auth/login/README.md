# AccessFlow Enterprise Portal

A secure employee verification and access management platform developed using Angular, TypeScript, Node.js and Express.js.

## Features

- Role Based Login
- Admin and General User Access
- Employee Dashboard
- Async Login Simulation
- Responsive Enterprise UI
- Modular Angular Architecture
- REST API Backend

## Technologies Used

- Angular
- TypeScript
- Node.js
- Express.js
- Angular Material
- Tailwind CSS
